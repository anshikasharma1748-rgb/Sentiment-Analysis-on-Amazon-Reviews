# Sentiment-Analysis-on-Amazon-Reviews
Sentiment analysis of Amazon product reviews using Machine Learning (Naive Bayes), Python, and Scikit-learn.
# Amazon Reviews Sentiment Analysis

## Project Overview
This project performs sentiment analysis on Amazon product reviews using Machine Learning. The model classifies customer reviews into Positive, Neutral, and Negative sentiments based on review ratings.

## Features
- Data Cleaning and Preprocessing
- Sentiment Classification
- Text Vectorization using CountVectorizer
- Machine Learning using Multinomial Naive Bayes
- Model Evaluation
- Sentiment Prediction
- Data Visualization

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Workflow
1. Load Amazon Reviews Dataset
2. Clean and preprocess the data
3. Create sentiment labels
4. Convert text into numerical features
5. Train the Naive Bayes model
6. Evaluate model performance
7. Predict sentiment of new reviews
8. Visualize sentiment distribution

## Output
- Positive, Neutral, and Negative sentiment prediction
- Accuracy Score
- Classification Report
- Confusion Matrix
- Sentiment Distribution Graph
